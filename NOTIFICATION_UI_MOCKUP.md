## Interaction Patterns

### Click Notification
- Marks as read
- Opens item detail page
- Highlights the item in inventory

### Swipe to Dismiss (Mobile)
- Swipe right: Mark as read
- Swipe left: Dismiss and snooze for 1 day

### Notification Actions
- "View Item" - Go to item details
- "Mark as Eaten" - Update status to EATEN
- "Discard" - Update status to DISCARDED
- "Snooze" - Remind again tomorrow

## Accessibility

- Screen reader announcements for new notifications
- Keyboard navigation (Tab through notifications)
- High contrast mode support
- Reduced motion option (disable animations)
- ARIA labels on all interactive elements
